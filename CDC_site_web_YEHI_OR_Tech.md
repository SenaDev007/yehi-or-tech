## 2. Contexte général du projet

Créer le site web officiel de **YEHI OR Tech**, une entreprise spécialisée dans les services numériques, le développement web/mobile, la conception graphique, les agents IA, l’automatisation métier, la présence digitale, la R&D en solutions informatiques et l’accompagnement à la transformation numérique.

Le site doit devenir une plateforme commerciale professionnelle, crédible, moderne et orientée conversion.

L’objectif n’est pas de créer un simple site vitrine basique. Le site doit être conçu comme un **outil d’acquisition client**, capable de :

- présenter clairement YEHI OR Tech ;
- valoriser ses services ;
- rassurer les prospects ;
- démontrer son niveau technique ;
- faciliter les demandes de devis ;
- orienter les visiteurs vers WhatsApp ou le formulaire de contact ;
- préparer l’évolution future vers un blog, un espace client, un portail projet ou une plateforme SaaS.

Le nom **YEHI OR** signifie : **“Que la lumière soit”**.

L’identité du site doit donc transmettre :

- la lumière ;
- la clarté ;
- l’innovation ;
- la technologie ;
- l’excellence ;
- la confiance ;
- la transformation digitale ;
- l’intelligence appliquée aux besoins réels des clients.

## 8. Stack technique exigée

Créer le site avec une stack moderne :

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**

Bibliothèques recommandées :

- `lucide-react` pour les icônes ;
- `framer-motion` pour les animations ;
- `next/image` pour les images ;
- `next/link` pour la navigation ;
- `react-hook-form` pour les formulaires ;
- `zod` pour la validation ;
- `clsx` ou `class-variance-authority` pour les classes conditionnelles ;
- `tailwind-merge` si utile.

Le site doit être :

- responsive ;
- mobile-first ;
- rapide ;
- SEO-friendly ;
- accessible ;
- modulaire ;
- maintenable ;
- prêt pour déploiement sur Vercel ;
- facilement extensible ;
- structuré avec des composants réutilisables ;
- alimenté par des fichiers de données centralisés.
